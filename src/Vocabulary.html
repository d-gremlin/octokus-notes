Git. How to work with

Git - это распределенная система контроля версий. Отслеживает изменения в репозитории. Рекомендуемый (годный) мануал по созданию Git.

Шпаргалка с основными командами для Git.

Основные команды подробнее

git commit - совершать, фиксировать. Git фокусируется на изменениях в файле, а не самом файле. Когда вы осуществляете команду git add file, вы не говорите git добавить файл в репозиторий. Скорее вы говорите, что git надо отметить текущее состояние файла, коммит которого будет произведен позже.
git commit - m - коммит с комментарием (message)
git push -
parcel - запуск сервера;
git add [файл]- проиндексировать(отметить) изменения в конкретном файле. Git add . - проиндексировать изменениях во всех файлах. Изменение записывается "ненавсегда".
git fetch- подгляди у соседа (origin), но ничего не делай. Вытягивает коммиты с информацией у удаленного репозитория, но пока ничего с ними не делает.
git pull- git fetch + git apply
git merge- git fetch + git
git diff- (namefile.diff) difference, сравнивает между предыдущим и текущим состоянием. Выявляет разницу и пишет какие строчки были добавлены или удалены
git stash- отложить на полку все текущие изменения и как эту полку назвать (откат до предыдущего коммита, добавить еще коммитов и накатить по-новой).
git cherry-pick- выборочно применяет изменений относитльено ветки master

SSH

Secure Shell(SSH) - (безопасная оболочка) это криптографический сетевой протокол для безопасной работы сетевых служб в незащищенной сети. Это протокол для общения между консолями.

SSH-keygen

SSH-keygen - SSH-ключи, используются для идентификации клиента при подключении к серверу по SSH-протоколу. Использовать этот способ вместо аутентификации по паролю.

Папка.ssh

Папка, где хранятся SSH-ключи.

Обозначения

Hyper Text Markup Language (англ. «язык гипертекстовой разметки») - HTML
1) h - heading - заголовок;
2) p - paragraph - параграф;
3) a - anchor - якорь;
4) href - hypertext reference - гипертекстовая ссылка;
5) target - мишень;
6) blank - чистый (новая вкладка);
7) src - source - источник;
8) alt - alternate - заместитель;
9) img - это тег. - image - изображение;
10) width - это атрибут - ширина;
11) br - перенос на новую строку (Enter);
12) b - жирный шрфит, /конец жирного шрифта;
13) Index - список, реестр, указатель. Проиндексировать изменения (git add) - "прикрепить указатель".
