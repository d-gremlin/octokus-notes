Git

Git - это распределенная система контроля версий. Отслеживает изменения в репозитории.

SSH

Secure Shell(SSH) - (безопасная оболочка) это криптографический сетевой протокол для безопасной работы сетевых служб в незащищенной сети. Это протокол для общения между консолями.

SSH-keygen

ssh-keygen - SSH-ключи, используются для идентификации клиента при подключении к серверу по SSH-протоколу. Используйте этот способ вместо аутентификации по паролю.

Папка.ssh

Папка, где хранятся SSH-ключи.
